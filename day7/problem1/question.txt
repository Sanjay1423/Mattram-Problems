Multiply and then sort an Array

Ramu wants to get N elements in an array and a number to be multiplied with the array elements.

He wants to display the multiplied result in descending order.

Help Ramu to complete this task.

Input Format

Number of array elements in the first line - N
N array elements in second line separated by a space
Number to be multiplied with array elements
Constraints

Positive Integers Only.

Output Format

Array elements in descending order after multiplied with the given number (space separated)

Sample Input 0

5
40 10 30 15
4
Sample Output 0

160 120 60 40 
Sample Input 1

5
10 30 20 50 40
2
Sample Output 1

100 80 60 40 20
