Pow Function 1

Implement the pow function using looping statement.

Input Format
First integer refers to the base
Second integer refers to the power

Constraints
NA

Output Format
Result calculated based on the given base and power

Sample Input 0
2 5

Sample Output 0
32

Explanation 0
2 x 2 x 2 x 2 x 2 = 32

Sample Input 1
6 3

Sample Output 1
216

Explanation 1
6 x 6 x 6 = 216
